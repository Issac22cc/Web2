Welcome

The purpose of this website is to help people who having amblyopia

There are five different parts in this website, you can click the link in the top right corner for the access of those parts , the following are the descriptions of different parts:

  1. Home Page: This is the home page which have an introduction of this website

  2. Photo Page: This is the first level of the practice , which contains some photos with the supports by audio description.

  3. Text Page: This is the second level of the practice, which contains some simple texts with the support by audio

  4. Paragraph Page: This is the third level of the practice, which contains some paragraphs with the support by audio

  5. Mini Game:This is the final level of the practice, which contain a mini game aim for helping amblyopia people
